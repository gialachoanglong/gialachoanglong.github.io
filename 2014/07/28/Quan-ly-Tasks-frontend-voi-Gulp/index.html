Gulp.js là gì



Khi làm việc với với frontend mình thường phải các việc như compile sass hay less thành css rồi gôm các file css riêng lẻ thành 1 file, gôm các thư viện js và các file js trong project lại thành 1 file hay là minify các file css , js hoặc image các việc này trước đây mình toàn phải tự làm và mình rất ghét điều này :( hiện tại thì việc này đã có nhiều tool để hỗ trợ mình làm như gulp và grunt mình không thích Grunt vì thấy nó config dài dòng phúc tạp quá nếu có nhiều stask thì sẽ rất dài và khó nhìn nên mình chọn Gulp

Cài đặt Gulp.js

cài đặt gulp ở chế độ global để sử dụng cho tất các project

nếu muốn chỉ cài gulp cho project hiện tại thôi thì:

cách này sẽ thêm vào phần devDependencies trong file package.json trong folder root của project.

Cài đặt các plugin cần thiết

  • Autoprefixer (gulp-autoprefixer)
  • Minify CSS (gulp-minify-css)
  • JSHint (gulp-jshint)
  • Concatenation (gulp-concat)
  • Uglify (gulp-uglify)
  • Compress images (gulp-imagemin)
  • LiveReload (gulp-livereload)
  • Clean files for a clean build (gulp-clean)
  • Compass sass compile (gulp-compass)

Tạo các tasks

Để tạo các task trước tiên cần tạo file gulpfile.js và mình gọi các plugin cần thiết và tạo các task trong file này. cách làm có thể xem trên gulp API docs
